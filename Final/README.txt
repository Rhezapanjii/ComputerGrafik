Nama Program : FlyingDrone
Penulis : Rheza Panji P N 
NIM : 211511056
Program ini menampilkan animasi dari Drone yang melayang diatas Gedung-gedung dan menampilkan objek matahari yang berotasi

Cara penggunaan :
1. Pastikan sudah mendowload apk Godot
2. Buka file Projek
3. Lalu run Projek dan klik tombol play
4. Maka akan tertampil main menu
5. Lalu pilih button Animasi 
6. Setelah itu akan tertampil animasi
7. Jika selesai lalu klik tombol back
8. Selesai
